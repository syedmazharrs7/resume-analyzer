# Resume Analyzer

A Python-based web application that analyzes resumes and provides ATS-style feedback to help candidates improve their profiles.

---

 Features :

- Upload resume in PDF format
- Extracts resume content automatically
- Calculates ATS-style match score
- Displays matched technical skills
- Provides personalized improvement suggestions

---

 Tech Used

- Python
- Streamlit
- PyPDF2

---

How It Works

1. User uploads a resume (PDF)
2. Text is extracted internally
3. Skills and sections are analyzed
4. ATS score is calculated
5. Suggestions are generated
